JSON

Sus siglas son de JavaScript Object Notation y es un formato ligero para almacenar y transportar datos, es fácil de leer, fácil de generar y está basado en una sintaxis muy similar a JavaScript, por lo que se convirtió en el estándar principal para comunicación entre aplicaciones web, hoy en día, JSON es el formato dominante en APIs modernas, reemplazando casi por completo a XML en la web.

Hoy en día a desplazado al XML teniendo funciones similares:

  • Intercambio de datos entre cliente y servidor
  • Configuración de aplicaciones
  • Almacenamiento de datos
  • Comunicación en tiempo real
  • Intercambio entre lenguajes distintos
  • APIs modernas (RESTful APIs)

Ejemplos de uso:

[ { "id": 1, "nombre": "Laptop HP", "precio": 600, "stock": 15 }, { "id": 2, "nombre": "Mouse Logitech", "precio": 20, "stock": 50 } ]
