XML

Sus siglas son de Extensible Markup Language y es un lenguaje de marcado diseñado para almacenar, estructurar y transportar datos, a diferencia de HTMLa que describe cómo se muestra la información, este describe la información en sí misma. Aunque hoy en día JSON es más común, XML sigue siendo esencial en muchas áreas del desarrollo web.

Este sirve hoy en día para:

  • Intercambio de datos entre sistemas
  • Configuración de aplicaciones web
  • Almacenamiento de datos estructurado
  • Validación de datos con esquemas
  • Transformaciones con XSLT
  • APIs y Web Services (especialmente SOAP)

Ejemplos de uso:
