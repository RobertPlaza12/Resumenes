AJAX

Sus siglas son de Asynchronous JavaScript And XML, hoy en día ya no depende solo de XML y puede usar JSON, texto o HTML, este permite que una página web se comunique con el servidor sin recargar la página completa, usando AJAX las aplicaciones web pueden ser más rápidas, interactivas y dinámicas.
Es una de las bases del comportamiento de aplicaciones modernas como lo pueden ser Facebook, Gmail, Twitter, paneles de administración, dashboards, etc.

Hoy en día nos sirve para:

  • Cargar datos sin recargar la página
  • Enviar formularios sin refrescar la web
  • Actualización parcial del contenido
  • Auto-completado y búsquedas en vivo
  • Validaciones en tiempo real
  • Cargar datos dinámicamente desde APIs

Ejemplos de uso:

fetch("/api/guardar", { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify({ nombre: "Randy", edad: 25 }) }) .then(res => res.json()) .then(data => console.log(data));
